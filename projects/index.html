Here you can find data and reproducible code for some of my projects. Feel free to use the code as it is or modify (and improve) it for your own purposes.

Please do not hesitate to get in touch if you encounter any errors.

Bibliometric analyses

Collaboration networks & open access (code and data for analyzing publications on Open Science from Dimensions; work in progress)

Gender & authorship (replication materials for analyzing gender differences in Computational Social Science publications from the Web of Science)

Crime analyses

History of death penalty (dataset and script for scraping, preprocessing, mapping, and visualizing historical data on executions in the United States, 1801-1900)

Serial killers (dataset containing scraped Wikipedia data on international serial killers and scripts for scraping, geocoding, visualizing, and mapping data on both international and US American serial killers)

Interactive dashboards

GitHub statistics (scraper and source code for a dashboard showing the Open Science MOOC's interactive collaboration network as well as repository statistics and user activities)

Project KillR (source code for a dashboard featuring data, statistics, and visuals on 576 serial killers from 51 countries for the years 1435 to 2013)

Natural language processing

Classic literature (script for preprocessing and analyzing public domain works using the example of Bram Stoker's Dracula)

Movie scripts (script for preprocessing and analyzing PDF files using the screenplay of The Room)

Music lyrics (script for preprocessing, analyzing, and visualizing STARSET album lyrics)

Network data

Renaissance Florence (datasets containing relational and attribute data on Florentine families for the years 1426-34)

Scraper functions

Amazon best sellers (scraping functions for 1. customer reviews from URLs and 2. product information of items from Amazon best seller lists)

Amazon customer reviews (scraper for all customer reviews for one or more Amazon products, with product ID/ASIN as input)

Aviation Safety Network (scraper for the aviation accident data provided in the Aviation Safety Network database; covers the years 1919-2019)

Plane Crash Info (scraper for the aviation accident data provided in the Plane Crash Info database; covers the years 1920-2019)

Social media analyses

Reddit: Today I Learned (script for scraping, preprocessing, and mining user comments from the TIL subreddit)

Social activism on Twitter (R Markdown document containing text and code for analyzing the development, key actors, and contents of the #We2 movement)

Twitter follower analysis (R Markdown replication script and data for analyzing the Open Science MOOC Twitter community)

Viewer engagement on YouTube (replication code and additional materials for analyzing YouTube comments and video statistics on the Florida High School Shooting in February 2018)

Spatial analyses

Airline flight routes (script for mapping airline routes using OpenFlights data in combination with NASA's night lights images)

Aviation accidents (replication materials for mapping and analyzing crashes in Florida 2014 with point pattern analyses) [script slightly outdated]

Road accidents (script for geocoding, mapping, and analyzing crashes in South Australia 2016)

Spatial gravity models (replication materials for building dyadic data sets and conducting predictive spatial gravity analyses)
